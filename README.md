# 📈 Upbit Volatility Breakout Trading Bot
> AWS EC2와 Python을 활용한 24시간 비트코인 자동매매 시스템

---

## 🚀 프로젝트 개요
변동성 돌파 전략(Larry Williams' Volatility Breakout Strategy)을 기반으로 업비트(Upbit)에서 비트코인을 자동 매매하는 시스템입니다. AWS 클라우드 환경에서 24시간 중단 없이 동작하도록 설계되었습니다.

## 🛠 기술 스택
- **Language:** Python 3.x
- **Infrastructure:** AWS EC2 (Ubuntu 22.04 LTS)
- **Database:** SQLite3 (매매 이력 및 수익률 추적)
- **Libraries:** pyupbit, pandas, python-dotenv

## ⚙️ 주요 기능
- **변동성 돌파 전략:** 실시간 시세 감시를 통한 목표가 돌파 시 즉시 매수
- **리스크 관리:** 고정 -3% 손절(Stop Loss) 및 고점 대비 -2% 익절(Trailing Stop) 로직 적용
- **데이터 영속성:** 모든 매매 발생 시 SQLite DB에 거래 내역 자동 기록
- **안정성:** `nohup`과 가상환경을 활용한 24시간 무중단 백그라운드 프로세스 운영
